## 📖 About

This repository contains my notes, labs, and practical exercises while learning Cyber Security.

The goal of this project is to document my progress from the basics of Linux and Web Security to solving real Capture The Flag (CTF) challenges.

Instead of only collecting payloads, I focus on understanding **how web technologies work**, **how vulnerabilities are discovered**, and **how they can be mitigated**.
